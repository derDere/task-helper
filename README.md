# TaskHelper

TaskHelper is a cross-platform productivity app built with [Flet](https://flet.dev) that helps you capture, prioritize and schedule tasks using an adaptive Elo-based ranking system. Instead of manually dragging tasks around, TaskHelper lets you quickly compare task importance pairwise; behind the scenes an Elo algorithm recalculates priorities so the most impactful tasks float to the top. A calendar view shows upcoming and completed work per day, while an organizer gives you fast access to adding tasks, sorting them, reviewing recently completed items, and browsing all tasks.

## Run the app

### uv

Run as a desktop app:

```
uv run flet run
```

Run as a web app:

```
uv run flet run --web
```

### Poetry

Install dependencies from `pyproject.toml`:

```
poetry install
```

Run as a desktop app:

```
poetry run flet run
```

Run as a web app:

```
poetry run flet run --web
```

For more details on running the app, refer to the [Getting Started Guide](https://flet.dev/docs/getting-started/).
